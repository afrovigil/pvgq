# Assets Publics (React / Vite)

Placez vos fichiers d'images ici (par exemple `notif.png` et `abmed.png`).
Vite copiera automatiquement tous les fichiers de ce dossier vers la racine du dossier de build `dist/` lors de la compilation (`npm run build`).

Ainsi :
- `public/notif.png` sera accessible via `src="notif.png"` dans l'application React.
- `public/abmed.png` sera accessible via `src="abmed.png"`.
